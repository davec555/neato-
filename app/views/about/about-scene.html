About the App
neato! was written and developed by Geoffrey Gauchet. Follow him on Twitter @zhephree.

The app uses PubSubHubBub technology to allow your web browser to push information to your phone. It uses a simple JavaScript API layer called PubNub.

Normally, this app would be free, but PubNub charges for every 5000 requests sent in a 24hr period. That will go quickly from everyday use, so the cost of this app helps pay for PubNub credits, as well as my hosting and bandwidth fees for the URL and bookmark generation.
